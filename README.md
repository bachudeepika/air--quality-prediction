# Air Quality Prediction

##  Project Description
This project predicts Air Quality Index (AQI) using a two-stage machine learning model.

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Features
- Classifies AQI categories (Good, Moderate, Poor)
- Predicts exact AQI values
- Uses real pollution data

## How to Run
1. Open Jupyter Notebook or Python file
2. Run all cells
3. View AQI prediction results
